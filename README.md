# 🧬 Virtual Molecular Diagnostics Laboratory

An interactive, single‑file educational web application that simulates molecular diagnostic techniques for first‑year medical biochemistry students. Built with vanilla HTML, CSS, and JavaScript.

---

## ✨ Features

* **Two interactive modules**
  - **Gel Electrophoresis** – Detect insertions and deletions by interpreting DNA band patterns.
  - **Restriction Fragment Length Polymorphism (RFLP)** – Identify point mutations that create or abolish restriction sites.
* **Realistic gel animation** – Run electrophoresis, watch bands migrate, and click them to see fragment sizes.
* **Clinical case generation** – Randomly generated cases with diseases like sickle cell anemia, cystic fibrosis, and more.
* **Scoring & progression** – Earn points, track streaks, and level up from *Laboratory Intern* to *Molecular Diagnostics Expert*.
* **Built‑in glossary** – Click on terms like “DNA”, “electrophoresis”, or “restriction enzyme” for instant definitions.
* **Responsive design** – Works on desktops, tablets, and phones.
* **Student feedback system** – Collects ratings, difficulty assessments, and written feedback directly into a Google Sheet.

## 📋 How to Use

1. **Home page** – Choose a module.
2. **Read the clinical case** and click **“Run Electrophoresis”** to start the animation.
3. **Examine the gel** – Hover over bands to see fragment sizes; click them for a quick toast.
4. **Interpret the results** by selecting the genotype for each patient.
5. **Submit your answers** to receive immediate feedback and earn points.
6. **Return to the home page** to try another random case or switch modules.
7. **Leave feedback** at the bottom of the home page to help improve the laboratory.
