# 🧬 Virtual Molecular Diagnostics Laboratory

An interactive educational web application designed to teach fundamental molecular diagnostic techniques to first-year medical students. The laboratory simulates real-world interpretation of DNA analysis using an engaging, case-based learning approach. Built entirely with vanilla HTML, CSS, and JavaScript as a single-file application.

---

## ✨ Features

### ⚡ Gel Electrophoresis Module
- Simulates DNA migration through agarose gel.
- Interpret insertions and deletions by analysing DNA band patterns.
- Interactive gel animation with clickable DNA bands showing fragment sizes.

### ✂️ Restriction Fragment Length Polymorphism (RFLP) Module
- Learn how restriction enzymes detect point mutations.
- Identify mutations that create or abolish restriction sites.
- Interpret normal, heterozygous, and homozygous mutant banding patterns.

### 🩺 Dynamic Clinical Cases
- Randomly generated patient cases for every session.
- Includes examples based on diseases such as:
  - Sickle Cell Anemia
  - β-Thalassemia
  - Cystic Fibrosis
  - Phenylketonuria
  - Familial Hypercholesterolemia
  - Tay-Sachs Disease
  - Gaucher Disease
  - Huntington Disease
  - and more.

### 🎮 Gamified Learning
- Earn points for correct interpretations.
- Progress through laboratory ranks:
  - 🧪 Laboratory Intern
  - 🔬 Junior Molecular Scientist
  - 🧬 Molecular Diagnostics Expert
- Immediate feedback after every answer.

### 📚 Interactive Glossary
Click scientific terms throughout the laboratory to view concise explanations without leaving the page.

### 📱 Responsive Design
Optimised for desktop, tablet, and mobile devices.

### 💬 Student Feedback
Built-in feedback form that allows students to:
- Rate the laboratory experience
- Assess the difficulty level
- Submit suggestions and comments

All responses are automatically stored in a connected Google Sheet using Google Apps Script for easy review by the instructor.

## 📋 How to Use

1. Open the Virtual Molecular Diagnostics Laboratory.
2. Select one of the two laboratory modules.
3. Read the clinical case provided.
4. Click **Run Experiment** to begin the simulation.
5. Observe the DNA fragments as they migrate through the gel.
6. Click DNA bands to view fragment sizes.
7. Interpret the patient's genotype by analysing the banding pattern.
8. Submit your answers to receive immediate feedback and earn points.
9. Generate a new case and continue practising.
10. Submit feedback at the bottom of the home page to help improve future versions of the laboratory.
